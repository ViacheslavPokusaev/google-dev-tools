1. Открыть dev tools и выбрать вкладку "Elements"
2. Для открытия поиска нажать Ctrl + F (рисунок 1)
3. 3 вида поиска: по строке (любой контент, любое название тега, скрипта и тд), CSS селекторы и XPath (https://soltau.ru/index.php/themes/dev/item/413-kratkoe-rukovodstvo-po-xpath)
- по контенту (рисунок 2)
- CSS селекторы (рисунок 3)
- XPath (рисунок 4)
4. Для переключения между результатами поиска, горячие клавиши:
- Вперёд: Ctrl + G (Enter или F3)
- Назад: Ctrl + Shirt + G
