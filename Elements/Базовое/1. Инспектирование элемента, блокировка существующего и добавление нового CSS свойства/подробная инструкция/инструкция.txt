1. Открыть DevTools нажатием клавиши F12 (also, Shift + CTRL + J)
2. Нажмите кнопку "Inspect element" вверху слева (рисунок 1). Аналогично для комбинации клавиш Shift + CTRL + C
3. Справа, на вкладке "Style" можно увидеть все примкнённые стили (рисунок 1)
4. Что бы отключить/добавить свойство смотрите рисунок 2
5. Для добавления "inline" свойств элементу, нужно перейти на вкладку Styles, секция "element.style" (рисунок 3). Пример на рис. 4
